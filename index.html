Facebook Page Reviews

Get reviews from a Facebook Page, using the Graph API Explorer.

Hi!
To get the reviews of a Facebook page you need to know the Page ID, have a Token, and some permissions.
You can have them at Graph API Explorer.

The data is parsed in JSON format, in an array with some reviews.
By example, you can access to:

  • "created_time"
  • "recommendation_type"
  • "review_text"
